# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/atharv-06/react-portfolio.git
```

Navigate to the project

```bash
cd react-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

# 📦 Build

```bash
npm run build
```

---

# 🚀 Deployment

Deploy to GitHub Pages

```bash
npm run deploy
```
